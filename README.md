# Maple Cake Gateway

## Requirements

* npm 5.x
* nodejs 9.x
* truffle 4.x

## Commands

Run unit tests:

```shell
npm run test
```

Compile contracts:

```shell
truffle compile
```

Install npm dependencies:

```shell
npm install
```

Start dev blockchain & deploy contracts:

```shell
truffle develop
migrate
```

Start development webserver:

```shell
npm run start
```

By default, the development server binds to `localhost:8080`.
